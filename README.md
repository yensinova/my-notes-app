✨✨  VUE 3 ✨✨ 

Aplicación desarrollada con el framework de javascript la cuál nos permite tener una aplicación 
conectada a una API para almacenar nuestras notas que vayamos creando. 

- Instalamos el router para tener navegación por las páginas y rutas; y pinia como gestor de
  estado global.
  ** npm install vue-router pinia **

- La carpeta store: nos permite gestionar recursos globales como peticiones o cualquier
  cosa a la API.
